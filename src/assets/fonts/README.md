# Bundled Fonts

## Latin Modern Math

`latinmodern-math.otf` is the OpenType math font from the CTAN `lm-math` package:

https://ctan.org/pkg/lm-math

The Latin Modern font family is distributed under the GUST Font License. Keep this note with the font asset when redistributing builds that bundle it.

## Latin Modern Roman

`lmroman10-regular.otf`, `lmroman10-bold.otf`, `lmroman10-italic.otf`, and `lmroman10-bolditalic.otf` are from the CTAN `lm` package:

https://ctan.org/pkg/lm

They are used as the matching text family for the native OpenMath playground mode and are also distributed under the GUST Font License.
